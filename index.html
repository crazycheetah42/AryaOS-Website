AryaOS

A better Ubuntu.





AryaOS is an Ubuntu-based distro (installed using a script) which provides significant improvements to the Ubuntu user interface and experience. With AryaOS, you get extra productivity apps, a faster system, a modern theme, and more! The script is also open-source on GitHub and anyone can add to it to improve the script.
